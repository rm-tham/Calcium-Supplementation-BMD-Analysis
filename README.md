# Calcium Supplementation BMD Analysis
This project examines the effect of calcium supplementation on bone mineral density in young adolescent girls under the age of thirteen. The longitudinal data is explored using data visualizations and LME models. Credits to writing this project go to marvelous Professor Koko Gulesserian. 

## Repository Contents
- **Longitudinal Analysis of Calcium Supplementation on Bone Mineral Density.pdf**: Contains interpreations of data visualizations and LME models.
- **calcium_supplementation_bmd_analysis.R**: Contains code creating data visualizations and LME models.
